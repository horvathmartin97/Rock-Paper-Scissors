Rock, Paper, Scissors Game
A browser-based Rock, Paper, Scissors game built with vanilla JavaScript. Play against the computer for 5 rounds with real-time scoring and input validation using browser dialogs.

✨ Features

    ✅ 5-round best-of game with live scoring
    ✅ Computer makes random, valid choices
    ✅ Input validation & case-insensitive matching
    ✅ Handles game cancellation gracefully
    ✅ Clear win/lose/tie feedback per round
    ✅ Uses native confirm() and prompt() dialogs


🎮 How to Play

    Open browser console (F12 → Console)
    Run the game: game()
    Confirm you're ready to play
    Enter rock, paper, or scissors each round
    Win 3+ rounds to beat the computer!
